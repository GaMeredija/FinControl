# Visao Geral

## Problema

Muitos usuarios nao conseguem entender para onde o dinheiro vai ao longo do mes. O sistema deve transformar lancamentos simples em informacoes de apoio a decisao.

## Objetivo do produto

Entregar uma aplicacao web onde o usuario consiga:

- registrar receitas e despesas
- organizar movimentacoes por conta e categoria
- acompanhar saldo e resumo mensal
- visualizar rapidamente excessos de gasto

## Publico-alvo

- estudantes
- profissionais em inicio de carreira
- pessoas que querem organizar a vida financeira sem planilhas

## Modulos principais

- Autenticacao
- Contas
- Categorias
- Lancamentos
- Dashboard

## Fluxo principal

1. Usuario cria conta e faz login
2. Usuario cadastra ao menos uma conta financeira
3. Usuario cadastra categorias ou usa categorias padrao
4. Usuario registra receitas e despesas
5. Sistema apresenta saldo, totais mensais e distribuicao por categoria

## Criterios de sucesso do MVP

- Usuario consegue entrar no sistema sem apoio manual
- Usuario consegue registrar movimentacoes do mes atual
- Dashboard reflete corretamente os dados cadastrados
- Dados de um usuario nunca aparecem para outro
