# Dados Locais

Os arquivos JSON desta pasta ficam versionados vazios no repositorio para o backend funcionar no modo `DATA_PROVIDER="json"` logo apos o clone.

Antes de publicar no GitHub:

- nao suba dados pessoais ou de teste
- mantenha os arquivos `*.json` como `[]` quando quiser um repositorio limpo
- use `backend/.env.example` como referencia, nunca o `backend/.env`

Durante a execucao local, o sistema preenche esses arquivos automaticamente.
